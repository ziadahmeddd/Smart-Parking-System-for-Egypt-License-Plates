# Model Files - Not Included in Git

Due to file size limitations, the following model files are **not included** in this repository:

## Required Model Files:

### 1. YOLO Models (Detection)
- **`plate_detector.pt`** - License plate detection model
  - Size: ~6MB
  - Used by: `maquette_main.py`, `smart_logger.py`
  
- **`character_detector.pt`** - Character recognition model
  - Size: ~6MB
  - Used by: `maquette_main.py`, `smart_logger.py`

### 2. TD3 Model (AI Allocation)
- **`td3_actor.pth`** - Trained parking allocation agent
  - Size: ~50KB
  - Generated by: `train_simulation.py`
  - Used by: `maquette_main.py` (via `td3_parking.py`)

### 3. Font File
- **`NotoSansArabic-Regular.ttf`** - Arabic text rendering
  - Size: ~400KB
  - Download from: [Google Fonts](https://fonts.google.com/noto/specimen/Noto+Sans+Arabic)


### Option 2: Download Pre-trained Models
Contact the repository owner for access to pre-trained models, or train your own YOLO models using the `dataset/` folder.

### Option 3: Use Your Own YOLO Models
If you have your own trained YOLO models for:
- Arabic/license plate detection
- Character recognition

Simply place them in the project root and update `config.py`:
```python
PLATE_MODEL_PATH = "your_plate_model.pt"
CHAR_MODEL_PATH = "your_char_model.pt"
```
